# sql-performance-tuning
Proyecto enfocado en la optimización de consultas SQL en SQL Server.

🎯 Objetivo

Mostrar cómo pequeñas mejoras en queries pueden generar grandes impactos en performance.

🧪 Contenido

Generación de dataset de prueba
Ejemplo de query no optimizada
Versión optimizada con índices

⚡ Resultados

Se demuestra cómo evitar funciones en columnas indexadas mejora significativamente los tiempos de ejecución.

🛠️ Tecnologías

SQL Server
T-SQL

📂 Estructura

scripts/: Consultas SQL
docs/: Análisis de resultados
